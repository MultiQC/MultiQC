The module parses results generated by HiCExplorere's hicBuildMatrix, a subtool to create an interaction matrix
out of mapped Hi-C reads.

### File search patterns

```yaml
hicexplorer:
  contents: Min rest. site distance
  max_filesize: 4096
  num_lines: 26
```

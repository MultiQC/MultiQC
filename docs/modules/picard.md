The Picard module parses results generated by
[Picard](http://broadinstitute.github.io/picard/),
a set of Java command line tools for manipulating high-throughput
sequencing data.

Supported commands:

* `MarkDuplicates`
* `InsertSizeMetrics`
* `GcBiasMetrics`
* `HsMetrics`
* `OxoGMetrics`
* `BaseDistributionByCycl`
* `RnaSeqMetrics`
* `AlignmentSummaryMetrics`
